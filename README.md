## ❓ Frequently Asked Questions

**Why does the system ask for network permissions?**
The software needs access to the network to talk to your playback devices. It does not look at your other internet activity.

**Can I run multiple broadcasts at the same time?**
Yes. The software manages several tasks at once. It automatically controls the flow to avoid network congestion.

**Where can I find the log files if something goes wrong?**
You can find logs in the "Help" menu. Select "View Logs" to see the history of your broadcasts. This helps you figure out why a transmission did not reach the destination.

**Does the software update automatically?**
The system checks for updates when you launch it. It lets you know if a newer version is available. Follow the screen prompts to update to the latest version.

**What happens if my connection drops?**
The application remembers the status of your broadcast. Once the network returns, the software picks up where it stopped. It ensures your media reaches the destination without missing pieces.
